WELCOME TO README!
If you have any problem, check out https://github.com/Adventures-of-Fox/FoxInstaller.

--- Info ---
    1. Minecraft server requires java 17
    2. Minecraft server requires minimal 4GB of ram (6GB IS RECOMMENDED) (8GB SHOULD BE MAX)
    3. Installer is made only for Linux and Window
    4. You need manually update modpack
--- Info ---

--- Linux ---
    1. Run linux.sh (if you cant run, type `chmod +x`)
    2. Wait to moment everything will be done
    3. Then run start.sh (edit RAM_MAX and RAM_MIN variables to change ram for server)
    4. Accept EULA
    5. Run again
    6. Done!
